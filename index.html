A Quick Guide On How to Set-up

Bower

  1. Install: npm install -g bower
  2. Check version: bower --v
  3. Generate bower.json file: bower init (press enter for default entry)
  4. Install packages to create bower_components: bower install (package_name like jquery, bootstrap, modernizr)
  5. Congratulations! You now got Bower packages!

Grunt

  1. Install: npm install -g grunt-cli
  2. Check version: grunt -version
  3. Generate package.json file: npm init
    Remove "main":"index.js"
  4. Manually include packages in package.json file, example:
    "devDependencies": {
        "grunt": "^0.4.5",
        "grunt-contrib-sass": "^0.7.3",
        "grunt-contrib-uglify": "^0.5.1"
    };
  5. Generate node_components: npm install --save-dev
  6. Manually create Gruntfile.js for the tasks
  7. Congratulations! Grunt is now set!

Sample Gruntfile.js

module.exports = function(grunt) {
//Project configuration

grunt.initConfig ({
pkg:grunt.file.readJSON('package.json'),
uglify: {
options: {
banner: '/*! <%= pkg.name %> <%= grunt.template.today("yyyy-mm-dd") %> */\n'
},
build: {
src: 'js/*.js',
dest: 'build/main.min.js'
}
},
sass: {
dist: {
files: {
'build/main.css':'sass/main.scss'
}
}
}
});

// Load the plugin that generates css from sass.
grunt.loadNpmTasks('grunt-contrib-sass');

// Load the plugin that provides the "uglify"
task. grunt.loadNpmTasks('grunt-contrib-uglify');

// Default task(s)
grunt.registerTask('default', ['uglify', 'sass']);
};
